Bright Harbor Dental

A practice project for a fictional dental practice — not a real client.

About this project

Bright Harbor Dental is a website built for a made-up dental practice, designed to show a full booking experience end to end. Visitors can browse services, book an appointment across one or more services, and receive a confirmation with a unique booking reference — with built-in protection against two people booking the same time slot.

Behind the scenes, the practice owner has a private, login-protected dashboard to review every booking and mark it Confirmed or Cancelled — access enforced at the database level, not just hidden in the interface.

  • React
  • Firebase (Firestore + Auth)
  • Vercel
