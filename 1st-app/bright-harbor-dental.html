About this project

Bright Harbor Dental is a full-stack booking platform for a fictional dental practice, covering the full lifecycle of a real appointment system — not just a booking form.

  • Real-time scheduling with built-in double-booking prevention, plus deposit payments via Stripe Checkout verified server-side through a signed webhook
  • Patient self-service rescheduling via a secure email link, staff-proposed time changes, and automatic refunds on early cancellation
  • A role-gated staff dashboard, with every permission enforced at the database level via Firestore Security Rules — not just hidden in the UI
  • Google Calendar sync and real email notifications at every step (booking, rescheduling, cancellation)
  • Production-grade security: rate limiting, CSP headers, CAPTCHA/App Check bot protection, and Sentry error monitoring
  • A full automated test suite (component, database-rules, and end-to-end tests) running in CI on every change
  • React
  • Firebase (Firestore + Auth)
  • Stripe
  • Vercel
